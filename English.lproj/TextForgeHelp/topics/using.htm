Using TextForge

You should find TextForge to be pretty straightforward. It was built to keep advanced features from cluttering up its interface, yet have them readily available to those who want to use them. Document windows are simple and uncluttered - most of the functionality is packed up in the menus. All commonly used commands have keyboard shortcuts. In order to maximize your TextForge experience, spend some time tweaking things in the preferences. If you have any ideas about how to improve TextForge's user experience, please let me know (josha@mac.com).
